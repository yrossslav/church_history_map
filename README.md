# Карта истории христианства

Интерактивная карта-генеалогия церквей и течений христианства — от апостольской Церкви
до наших дней. Видны происхождение каждой ветви, ключевые даты, эпохи и связи; у каждого
блока — описание и ссылки на источники.

**Открыть онлайн:** https://yrossslav.github.io/church_history_map/
(после включения GitHub Pages — см. ниже)

## Что внутри
- `index.html` — страница
- `styles.css` — оформление
- `app.js` — движок (пан/зум, карточки, источники, залипающие шапка и таймлайн, эпохи)
- `data.js` — данные карты (узлы, связи, события, источники)

Это полностью статический сайт: никакого сервера не нужно — можно открыть `index.html`
двойным кликом или опубликовать на GitHub Pages.

## Управление
- Колесо — зум, перетаскивание — двигать (как в Miro).
- Наведение на блок — увеличение; клик — описание и подсветка ветви.
- Левый таймлайн — даты (наведение — событие, клик — подробно).
- Меню (☰) справа — слои и «Все источники».

## Этапы
Заголовки меняются по трём эпохам: I — Древняя Церковь (до 1054),
II — Разделённая Церковь (1054–1517), III — Реформация → наши дни.

## Источники
Содержание собрано по учебнику истории Реформации (Двуреченский, рукопись),
С. В. Санникову «Двадцать веков христианства», Ф. Шаффу, N. Needham
«2000 Years of Christ's Power» и др. Полный список — в карточках («Все источники»).

## Лицензия / авторство
Учебные материалы. При использовании указывайте источники.
